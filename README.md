# WEP-PROGECT

## YOLO옵서예
* 2040 세대들 즉 젊은 층을 타겟으로 설정하여 제주도의 각 구역별 먹거리, 놀거리, 볼거리에 관한 정보를 소개해주는 클라이언트 사이트


### 관광지 데이터 선정의 기준?
* 타겟을 2040으로 설정한 만큼 젊은 층의 사용자가 많은 인스타그램 해시태그 수 많은 순으로 선정
* 타겟 사이트인 VISIT 제주 사이트에서 리뷰 수 높은 순으로 선정
